## Requirements

- Python 3.11 (newer versions may not fully support mediapipe)
- Dependencies:
  - opencv-python
  - mediapipe
  - numpy
  - pandas
  - matplotlib
  - pytube

## Setup

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python main.py`
